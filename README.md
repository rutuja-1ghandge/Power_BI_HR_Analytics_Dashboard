# Power_BI_HR_Analytics_Dashboard

this project presents an HR Analytics Dashboard created using Power BI to analyse empolyee attribition, salary, age, job role distibution nad other KPIs . The dashboard helps HR 
team to understandt workforce pattern and make data-driven decisions.

## Key Insights

Total Employees: 961
Total Attrition: 133
Attrition rate: 13.8%
Average age: 37
Average salary: 6.3k
Average age at company: 6.9

## Dashboard visuals
The dashboard visuals include the following visuals

1. Attrition by Gender
   Bar chart showing male vs female Attrition coun.

2. Attrition by Education field
   pie chart attrition across:
   - Medical
   - Life Science
   - Technical Degree
   - Human Resources
   - Other

3. Attrition by Age Group
   Bar chart showing employees leaving in age groups

   18-25, 26-35, 36-45, 46-55, 55+

4. Attrition by Department

   -HR
   -Research & Development
   -Sales

5. Attrition by salary slab

   Visualization based on salary ranges (e.g., <5k,5-10k, etc.)

6. Attrition by job Role

   Matrix showing job roles vs attribution count.

7. Attrition Trend(Line chart)
   Monthly/overall attrition trend.
